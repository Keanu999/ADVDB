### Install  BACKEND
```

npm install
```

### Install  FRONTEND


```
cd frontend
npm install
cd ..
```

### Start  App

# Run frontend & backend at the same time


```
npm run dev
```



## Build + Deploy

# Create frontend  build
```
cd frontend
npm run build
```
